Data Handling 1 - Glossary

Glossary DH1
pandas
Pandas is a Python library. Pandas is used to store, manipulate and analyse data inside its proprietary DataFrame, a tabular data structure akin to a spreadsheet.
DataFrame
A 2-dimensional labeled data structure with columns of potentially different types. You can think of it like a spreadsheet or SQL table. It is the core data structure in Pandas.
CSV (Comma-Separated Values)
A file format that stores tabular data in plain text, where each line represents a row and values are separated by commas. Commonly used for data exchange and can be read with `read_csv()`.
Index
The labels on the left side of a DataFrame that identify rows. By default, this is sequential integers starting from 0, but can be customised to use any labels.
Columns
The labels across the top of a DataFrame that serve as headers for each column. They form part of the DataFrame's metadata.
shape
A DataFrame property that returns a tuple containing the number of rows and columns (rows, columns).
dtypes
A DataFrame property that shows the data type of each column (e.g., int64, float64, object).
Slicing
.loc
.iloc
(deprecated) Integer position-based indexing method that uses numerical index positions. As of Pandas 2.2.0, `.loc` is preferred.
NaN (Not a Number)
Python's standard way of representing missing or null values in data. Common in real-world datasets.
Series
A one-dimensional labeled array in Pandas. When you select a single column from a DataFrame or apply methods like `.mean()`, you get a Series.
drop()
A DataFrame method used to remove rows or columns. Use `columns=` parameter to specify columns to drop and `inplace=True` to modify the DataFrame directly.
Filtering
The process of selecting rows from a DataFrame that meet specific conditions using boolean expressions (e.g., `df[df['column'] > 10]`).
Boolean indexing
Using True/False conditions to filter DataFrame rows. The condition returns a boolean Series that acts as a mask.
Matplotlib
A comprehensive library for creating static, animated, and interactive visualisations in Python. Used extensively for data visualisation.
fig
The overall figure object in Matplotlib that contains all plot elements. Used for figure-level settings like `suptitle()` and `tight_layout()`.
ax
The axes object in Matplotlib representing a single plot or coordinate system where data is plotted.
Boxplot
A visualisation that displays the distribution of data through quartiles, showing median, 25th/75th percentiles, and outliers.
Histogram
A plot that shows the frequency distribution of data by dividing the range into bins and counting values in each bin.
Bins
The containers or intervals used in a histogram to group data values. The number of bins affects the granularity of the distribution display.
Alias
A shorthand name for an imported library (e.g., `import pandas as pd`), making it easier to reference in code.
Property
An attribute of an object that doesn't require parentheses to access (e.g., `df.shape`, `df.columns`), as opposed to methods which do.
Method
A function that belongs to an object and is called with parentheses (e.g., `df.head()`, `df.describe()`).
Metadata
Data about the DataFrame structure (index, column names) that is not part of the actual data values.
Univariate analysis
Analysis of a single variable/column independently, examining its distribution and characteristics.
Bivariate analysis
Analysis examining the relationship between two variables/columns.
Outliers
Data points that differ significantly from other observations, often visible in boxplots as points beyond the whiskers.
NumPy
A fundamental Python library for numerical computing that provides support for arrays and mathematical operations. Pandas is built on top of NumPy.
Relative path
A file path specified relative to the current working directory (e.g., `./data/file.csv`).
Absolute path
A complete file path from the root directory (e.g., `/Users/username/data/file.csv`).
